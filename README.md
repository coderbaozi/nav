## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`Nodejs 16+`
`npm`


## Installation

Install my-project with npm

```bash
  git clone https://github.com/coderbaozi/baozi-Navigation.git
  cd baozi-Navigation
  npm install
  npm run dev
```
